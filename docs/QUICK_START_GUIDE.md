## ‚è±Ô∏è 5-Minute Tutorial

### What You'll Learn
- Create a report from a template
- Customize widgets
- Apply filters
- Export to PDF

### Prerequisites
- At least one survey with responses
- 5 minutes of time

### Let's Begin!

#### 1. Create Report from Template (1 minute)

1. Click the **"Create Report"** button (top right)
2. Select **"Use a Template"**
3. Choose your survey from the list
4. Browse the template gallery
5. Click **"Use This Template"** on "NPS Overview Dashboard"
6. Name your report (e.g., "February NPS Analysis")
7. Click **"Create"**

‚úÖ **Result**: You now have a complete NPS dashboard!

#### 2. Explore Your Report (1 minute)

Your new report includes:
- **KPI Card**: Overall NPS score with trend
- **Trend Chart**: NPS over time
- **Distribution Chart**: Promoters, Passives, Detractors breakdown
- **Word Cloud**: Common themes from comments
- **Data Table**: Recent responses

**Try This**:
- Hover over charts to see details
- Click column headers in the table to sort
- Use mouse wheel to zoom charts

#### 3. Apply a Filter (1 minute)

Let's filter to see only recent data:

1. Click **"Filter"** button in the toolbar
2. Select field: **"Response Date"**
3. Choose operator: **"Between"**
4. Enter dates: Last 30 days
5. Click **"Apply"**

‚úÖ **Result**: All widgets now show only the last 30 days of data!

**Filter Badge**: Notice the blue badge next to the Filter button showing "1 active filter". Click it anytime to remove the filter.

#### 4. Customize a Widget (1 minute)

Let's change the chart type:

1. Click on the **Trend Chart** to select it
2. The **Properties Panel** opens on the right
3. Under **"Chart Type"**, select **"Area"**
4. Under **"Colors"**, choose a different color
5. Click anywhere outside to deselect

‚úÖ **Result**: Your trend chart is now an area chart with custom colors!

#### 5. Export to PDF (1 minute)

Let's save your report as PDF:

1. Click **"Export"** button in toolbar
2. Select **"PDF"** format
3. Choose **"Landscape"** orientation
4. Check **"Include Charts"**
5. Uncheck **"Include Data"** (for a cleaner report)
6. Click **"Export"**

‚è≥ Wait 5-10 seconds for generation...

‚úÖ **Result**: PDF automatically downloads! Open it to see your professional report.

## üìä Create Your First Report

### Option A: Quick Start with Template (Recommended)

**Best for**: Getting started quickly, learning by example

**Steps**:
1. **Choose Template**: Browse 20+ pre-built templates
2. **Select Survey**: Pick which survey to analyze
3. **Create**: One-click report generation
4. **Customize**: Modify to fit your needs
5. **Save**: Your report is ready!

**Popular Templates**:
- **NPS Overview Dashboard**: Complete NPS analysis
- **Customer Satisfaction**: CSAT metrics and trends
- **Response Analysis**: Detailed breakdown
- **Delivery Performance**: Multi-channel tracking
- **Sentiment Trends**: Opinion analysis

**Time**: 2-3 minutes

### Option B: Start from Scratch

**Best for**: Custom requirements, advanced users

**Steps**:

#### 1. Open Designer
1. Click **"Create Report"**
2. Select **"Start from Scratch"**
3. Choose your survey
4. Name your report
5. Click **"Create"**

#### 2. Add Widgets
1. Open **"Visuals Gallery"** (right panel)
2. Browse widget types:
   - **KPI Card**: Single metric
   - **Chart**: Bar, line, pie, etc.
   - **Table**: Data grid
   - **Cohort Analysis**: User segments
   - **Forecast**: Predictions
   - **Key Drivers**: Impact analysis
   - **Word Cloud**: Text visualization
3. Drag widget to canvas
4. Widget appears in grid

#### 3. Configure Widget
1. Click widget to select
2. Properties Panel opens
3. Configure:
   - **Title**: Widget name
   - **Data Source**: Which question/field
   - **Chart Type**: Visualization style
   - **Aggregation**: How to calculate
   - **Filters**: Widget-specific filtering
   - **Colors**: Customize appearance

#### 4. Arrange Layout
- **Move**: Drag widget to reposition
- **Resize**: Drag corners to resize
- **Delete**: Click X button on widget
- **Duplicate**: Right-click ‚Üí Duplicate

#### 5. Save Report
- Click **"Save"** in toolbar
- Report auto-saves while editing
- Close designer to return to list

**Time**: 10-15 minutes

## üìù Common Tasks

### View All Reports

**Path**: Analytics Studio ‚Üí Reports List

Shows all your reports with:
- Thumbnail preview
- Title and description
- Last modified date
- Actions (open, edit, delete)

**Tips**:
- Use search to find reports quickly
- Sort by date, name, or recent
- Star favorites for quick access

### Open a Report

**Methods**:
1. Click report card in list
2. Use search and click result
3. Bookmark direct URL

Opens in **viewer mode** with all widgets displayed.

### Edit a Report

**From List**:
1. Hover over report card
2. Click **"Edit"** button

**From Viewer**:
1. Click **"Edit"** button in toolbar

Opens in **designer mode** where you can modify layout and widgets.

### Duplicate a Report

Create a copy to use as a starting point:

1. Open report
2. Click **"..."** (More menu)
3. Select **"Duplicate"**
4. Name your copy
5. Click **"Create"**

### Delete a Report

1. Hover over report card
2. Click **"Delete"** button (trash icon)
3. Confirm deletion

‚ö†Ô∏è **Warning**: Deleted reports cannot be recovered!

### Apply Global Filters

Filters that affect the entire report:

1. Open report in viewer or designer
2. Click **"Filter"** button
3. Choose field to filter
4. Select operator and value
5. Click **"Apply"**

**Active filters** show as badges. Click badge to remove.

### Change Date Range

Quick way to filter by date:

1. Click **"Date Range"** dropdown (if available)
2. Select preset:
   - Last 7 days
   - Last 30 days
   - Last 90 days
   - Year to date
   - Custom range
3. Data refreshes automatically

### Export a Report

**Formats**: PDF, PowerPoint, Excel, CSV

**Steps**:
1. Open report
2. Click **"Export"** button
3. Select format
4. Configure options
5. Click **"Export"**
6. Wait for generation
7. Download automatically starts

**Options by Format**:

**PDF**:
- Orientation: Portrait/Landscape
- Include charts: Yes/No
- Include data: Yes/No
- Page size: A4/Letter/Legal

**PowerPoint**:
- Title slide: Yes/No
- Charts per slide: 1 or 2
- Theme: Light/Dark

**Excel**:
- Full data or summary only
- Multiple sheets
- Formulas included

**CSV**:
- Raw data export
- For external analysis tools

### Schedule a Report

Automatically generate and email reports:

1. Open report
2. Click **"Schedule"** button
3. Configure:
   - **Frequency**: Daily, Weekly, Monthly
   - **Time**: When to send (e.g., 9:00 AM)
   - **Day**: For weekly/monthly
   - **Recipients**: Email addresses
   - **Format**: PDF or PowerPoint
4. Click **"Create Schedule"**

**Manage Schedules**:
- View all: **Scheduled Reports** tab
- Edit: Click schedule ‚Üí **Edit**
- Pause: Toggle **Active** switch
- Delete: Click **Delete** button

### Use Specialized Dashboards

**Survey Analytics**:
1. Click **"Survey Analytics"** tab
2. View response overview, trends, question analysis

**Delivery Performance**:
1. Click **"Delivery Performance"** tab
2. See multi-channel delivery metrics and funnel

**Sentiment Analysis**:
1. Click **"Sentiment Analysis"** tab
2. Track opinion trends and sentiment distribution

**AI Insights**:
1. Click **"AI Insights"** tab
2. Ask questions in natural language
3. View automated insights and recommendations

## üí° Tips & Tricks

### Tip 1: Start with Templates
Templates are a great way to learn. Create from template, then customize!

### Tip 2: Use Filters Liberally
Narrow your data to focus on what matters. You can always remove filters later.

### Tip 3: Name Reports Clearly
Use descriptive names like "Q1 2026 NPS Analysis" instead of "Report 1".

### Tip 4: Save Often
Reports auto-save, but click Save after major changes for peace of mind.

### Tip 5: Export for Sharing
PDFs and PowerPoints are great for presentations and email.

### Tip 6: Schedule for Automation
Set up weekly/monthly schedules for recurring reports.

### Tip 7: Use Keyboard Shortcuts
Learn shortcuts like `Ctrl+S` (save) and `Ctrl+E` (export) for faster workflow.

### Tip 8: Explore Specialized Dashboards
Don't miss the Delivery Performance and Sentiment Analysis dashboards!

### Tip 9: Check Performance
Use the floating toggle button to compare legacy vs new version performance.

### Tip 10: Ask Questions
The AI Insights dashboard lets you ask questions in plain English!
