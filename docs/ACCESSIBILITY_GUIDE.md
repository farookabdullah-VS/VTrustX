# Analytics Studio - Accessibility Guide

## WCAG 2.1 Level AA Compliance

The Analytics Studio meets WCAG 2.1 Level AA standards for accessibility.

## Keyboard Shortcuts

### Global
- Alt + M: Jump to main content
- Alt + N: Jump to navigation  
- ?: Show keyboard shortcuts

### Widget Actions
- Ctrl + E: Edit widget
- Ctrl + Delete: Remove widget
- Ctrl + F: Expand fullscreen

### Report Actions
- Ctrl + S: Save report
- Ctrl + P: Print report

## Features

- Full keyboard navigation
- Screen reader support (NVDA, JAWS, VoiceOver)
- ARIA labels on all elements
- High contrast mode support
- Touch targets >= 44px

## Testing Tools

- axe DevTools
- WAVE
- Lighthouse
- NVDA screen reader
