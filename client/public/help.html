User Manual

Welcome to RayixCX, the intelligent platform for creating, managing, and analyzing customer experiences.

üöÄ Getting Started

Accessing the Platform

  • Web URL: https://rayixcx.web.app
  • Login: Use your Administrator credentials.

Once logged in, you will be taken to the main Dashboard where you can see recent activity and key metrics.

üìä Survey Management

RayixCX makes it easy to create beautiful surveys in minutes.

Create New

Click + New Survey to start. Choose from a Blank Canvas, Templates (NPS, CSAT), or use the AI Generator to build one from a prompt.

Import

Already have questions? Upload an Excel or CSV file to instantly generate your survey structure.

Edit & Logic

Use the Drag-and-Drop editor to add questions. Configure skip logic to show/hide questions based on answers.

üé® Theme & Customization New

Ensure your surveys match your brand identity perfectly.

The Theme Engine

Located in the Survey Editor (Palette Icon), you can control:

  • Theme Library: One-click presets like RayiX Modern, VTrust Legacy, and the new Vision Bank (Dark Future).
  • Typography: Set distinct fonts for English (LTR) and Arabic (RTL) interfaces.
  • Navigation: Choose button styles (Square, Pill) and icons (Arrow, Hand, Chevron).
  • Backgrounds: Add dynamic patterns like Polka dots or Gradients.

‚öñÔ∏è Quotas & Response Limits New

Control the flow of incoming data to meet your study requirements.

Setting Limits

Go to Settings > Response Quotas.

  1. Global Limit: Set a maximum number of responses for the entire survey.
  2. Segmented Quotas: Click + Add Quota to define rules.
    Example: Limit "Male" respondents to 50.

Behavior: When a quota is reached, any new attempt to submit that matches the criteria will be rejected with a user-friendly "Survey Closed" message.

ü§ñ AI Features

  • Voice Agent: Enable a conversational AI that interviews users via voice instead of text forms.
  • Sentiment Analysis: Automatically tag open-ended responses with sentiment (Positive/Negative) and topics.

üíª Developer Guide (SDK)

Embed RayixCX surveys on your own website using our lightweight Javascript SDK.

Installation

Add this script tag to your site:

<script src="https://rayixcx.web.app/rayix-sdk.js"></script>

Initialization

RayixCX.init({
    slug: 'YOUR_SURVEY_SLUG',
    mode: 'modal', // Options: 'modal', 'inline', 'button'
    buttonText: 'Give Feedback',
    themeColor: '#5B4DFF'
});
