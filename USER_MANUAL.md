## âš–ï¸ Quota Management (Response Limits)

Control how many responses your survey collects to stay within budget or target.

### Setting Limits
1. Open a Survey and go to **Settings** > **Response Quotas**.
2. **Global Form Limit**: Set a hard limit on total responses (e.g., 1000).
3. **Specific Quotas (Segments)**:
   - Click **"+ Add Quota"**.
   - **Label**: Give it a name (e.g., "Male Respondents").
   - **Limit**: Set the count (e.g., 50).
   - **Criteria (JSON)**: Define the rule.
     - *Example*: `{"gender": "male"}`
     - *Leave empty* to apply to all responses (Global Quota).

### What happens when a quota is met?
- New respondents will see a **"Submission Rejected"** screen explaining that the quota has been reached.
- The system will block the submission to ensure data integrity.

## ðŸ’» Developer Guide: Javascript SDK

Embed RayixCX surveys directly into your website or mobile app.

### Installation
Add the script to your HTML `<head>` or `<body>`:
```html
<script src="https://rayixcx.web.app/rayix-sdk.js"></script>
```

### Usage
Initialize the SDK with your survey details:

```javascript
RayixCX.init({
    slug: 'YOUR_SURVEY_SLUG', // Found in Survey Settings
    mode: 'modal',            // 'modal', 'inline', or 'button'
    buttonText: 'Give Feedback',
    themeColor: '#5B4DFF',
    position: 'bottom-right'
});
```

### Modes
- **Modal**: Shows a floating button. Clicking it opens a popup overlay.
- **Inline**: Embeds the survey inside a `div` on your page (requires `containerId`).
